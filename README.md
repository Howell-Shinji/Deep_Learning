# PyTorch 深度学习练手项目
## 项目简介
这是一个基于 PyTorch 的深度学习练手项目，旨在帮助初学者熟悉 PyTorch 的基本使用方法，掌握构建、训练和测试神经网络的流程。本项目涵盖了常见的深度学习任务，如图像分类、回归预测等，通过实践帮助学习者加深对深度学习理论的理解。
项目结构:
```
PyTorch-Practice-Project/
├── models/                 # 模型文件
├── README.md               # 项目说明文档
```

## 环境依赖
- Python 3.10
- PyTorch 1.10+
- torchvision
- numpy
- matplotlib
- tqdm
- cuda
推荐使用Anaconda进行pyton环境管理。
